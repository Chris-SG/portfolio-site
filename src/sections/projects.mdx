## Projects

<ProjectCard title="Window Hide" lang="C#" link="https://github.com/Chris-SG/WindowHide" bg="linear-gradient(to left, #4b6cb7, #182848)">
    Watchdog used to automatically maximise, minimize or hide windows based on title.
</ProjectCard>

<ProjectCard title="File Mover" lang="C++" link="https://github.com/Chris-SG/file_mover" bg="linear-gradient(to right, #4cb8c4, #3cd3ad)">
    Distribute directory to various destinations. Supports both local directories and remote shares.
    This project involved working around many pitfalls relating to the Windows API, specifically in regards to authentication.
    Includes a basic logger with time formatting.
</ProjectCard>

<ProjectCard title="File Watch" lang="C++" link="https://github.com/Chris-SG/file_watch" bg="linear-gradient(to right, #d53369, #cbad6d)">
    Watch a given directory. When any files are added, instantaneously move them to another directory. Supports remote destinations with authentication.
</ProjectCard>

<ProjectCard title="practosu!" lang="C++" link="https://github.com/Chris-SG/practosu" bg="linear-gradient(to left, #8e2de2, #4a00e0)">
    A practice toolkit for the rhythm game 'osu!'.
    Written using the QT framework, this project involves binary parsing, memory scanning and more.
    Development on this tool has halted in favour of a new API written from scratch, still in progress.
</ProjectCard>

<ProjectCard title="Lambda Calculus Language" lang="Java" link="https://github.com/Chris-SG/LambdaCalculusLanguage" bg="linear-gradient(to right, #77a1d3, #79cbca)">
    A Java-based parser of lambda statements.
    Including both the parser as well as an abstract syntax tree, evaluates the validity of lambda expressions.
</ProjectCard>

<ProjectCard title="Bauxe Bot Go" lang="GoLang" link="https://github.com/Chris-SG/BauxeBot_Go" bg="linear-gradient(to right, #0f0c29, #302b63)">
    A basic Discord chatbot developed as an introduction to the Go language.
    Interacts with the Discord API to perform various actions based on command types.
</ProjectCard>

<ProjectCard title="Maze Navigator" lang="C++" link="https://github.com/Chris-SG/MazeNavigator" bg="linear-gradient(to left, #e96443, #904e95)">
    An AI built to solve mazes.
    Supports A*, Breadth-First, Depth-First, Greedy-Best-First and Iterative-Deepening-Depth-First searches.
    Additionally contains a basic timer, logger and image output.
</ProjectCard>
